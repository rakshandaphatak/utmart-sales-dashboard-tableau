# 📊 UT MART Sales Analysis | Tableau Dashboard

## 📌 Project Description
This project focuses on analyzing sales performance, customer purchasing behavior, and category growth trends using Tableau.
---

# 🎯 Objectives

- Analyze overall sales performance and trends.
- Identify high-performing categories and sub-categories.
- Evaluate regional sales contribution.
- Understand customer ordering patterns.
- Generate actionable recommendations.

---


# 📊 Key Takeaways

✅ Technology is the top revenue contributor.

✅ Regional strategies can improve performance.

✅ Product optimization can increase profits.

✅ Forecasting supports inventory planning.

---

# 💡 Business Recommendations

### 1. Expand Technology Sales
Increase promotions and inventory.

### 2. Improve Average Order Value
Use:
- Product Bundles
- Cross Selling
- Discounts

### 3. Strengthen Weak Regions
Run targeted campaigns.

### 4. Optimize Product Mix
Focus on high-performing categories.

### 5. Forecast Demand
Plan inventory using trends.

# 🚀 Business Impact

This dashboard helps:

- Increase profitability

# Conclusion

This Tableau dashboard provides a complete business performance overview and supports data-driven decision-making.
